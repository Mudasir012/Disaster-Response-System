# Product

## Register

product

## Users

Emergency responders — professional disaster response teams monitoring incidents in real-time under pressure. They need to assess situations quickly, filter noise from signal, and make decisions with confidence. Usage spans command centers (large monitors, low ambient light) to field tablets (outdoor glare, limited attention).

## Product Purpose

Real-time global disaster monitoring and alerting platform powered by AI. Aggregates live data from authoritative sources (USGS, GDACS, NOAA, GDELT), classifies using Google Gemini AI, displays on an interactive map, and sends instant alerts. Success means responders see what matters before anyone else does.

## Brand Personality

Calm, Clear, Authoritative. Reassuring presence during crisis. Voice is precise and professional — never sensationalist, never alarmist. Data speaks; the interface amplifies, never decorates.

## Anti-references

- No clichéd dashboard templates — busy widgets, floating charts, unnecessary gradients
- No sensationalism — no alarmist disaster aesthetics, pulsing red panic, or dramatic typography
- No "hero-metric" patterns — big numbers with tiny labels feel like marketing, not operations
- No glassmorphism as decoration — functional blur for overlays only, never cards

## Design Principles

**Clarity under pressure.** Every decision prioritizes readability and quick comprehension during stressful situations. High contrast, generous type, unambiguous hierarchy. If it slows a responder down by half a second, it doesn't belong.

## Accessibility & Inclusion

- WCAG 2.1 AA compliance target for all interfaces
- High-contrast mode considerations for outdoor and bright-light field use
- Keyboard-navigable map interactions
- Color is never the sole differentiator — shape, position, and text accompany severity signals
